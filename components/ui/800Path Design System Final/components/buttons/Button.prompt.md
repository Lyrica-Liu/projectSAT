Pill-rounded brand button — use for every primary/secondary call to action.

```jsx
<Button variant="primary" size="md" onClick={start}>Start session</Button>
```

Variants: `primary` (lilac, the hero action), `secondary` (outlined white), `ghost` (text-only, low emphasis), `soft` (pale lilac fill). Sizes `sm | md | lg`. Pass `full` to stretch, `iconLeft`/`iconRight` for inline icons, `disabled` to dim. Press scales to 0.97 — no bounce.
