The core surface — soft white, rounded 16px, hairline border, low shadow.

```jsx
<Card tone="surface" padding="lg" interactive>…</Card>
<Card tone="brand">Hero CTA</Card>
```

`tone`: `surface` (default white), `sunken`, `brand` (filled lilac), or pastel `lilac | mint | sky | rose`. Control `padding`, `radius`, `shadow`. Set `interactive` to lift on hover. Use `brand` for the dashboard "Start a session" hero.
